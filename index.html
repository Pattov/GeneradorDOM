¿Que se puede hacer en DOM?

Se puede seleccionar, modificar, eliminar o agregar de elementos en la página web.

¿No funciona tu proyecto?

Acuerdate de poner en el body del HTML

<div id="padre"><!-- Aquí se agregarán los elementos --></div>

En el JS // hay que obtener el elemento padre

const padre = document.getElementById("padre");

¿Como hacer un elemento invisible?

display: none → desactiva el diseño de los elementos, evitando que se represente

elemento.style.display="none";

visibility: hidden → oculta los elementos sin cambiar su diseño

elemento.style.visibility="hidden";

opacity: 0 → los elementos pasan a transparentes, pero se puede interactuar con ellos.

elemento.style.opacity="0";

¿Como hacer un Evento?

Tiene dos partes importantes, el nombre del evento y la funcion que maneja el evento

padre.addEventListener("click", ()=>{ console.log("Hola");})

Tambien se puede usar otras funciones como removeEventListener()

Hay otros eventos como de ratón (dblclick, mouseover, mouseout, mousedown y mouseup), de teclado (keydown, keyup y keypress), de formularios (submit, focus, blur y input) y de ventana (load, resize y scroll)
